## v2.1.1 (2024-12-01)

### ✨ 新功能

**1. 待办图片上传**
- 创建/编辑待办时支持上传图片附件
- 支持图片预览和删除
- 图片存储在本地数据目录

**2. 深色主题透明模式**
- 仅在深色主题下可用
- 透明度滑块：1% ~ 100% 可调
- 实时预览，切换主题自动重置
- 纯 CSS 透明，无系统级模糊干扰

**3. 清除缓存功能**
- 一键清除所有使用痕迹
- 支持选择性清除：待办数据 / 设置 / 本地存储
- 操作前二次确认，防止误删

### 🎨 UI 优化

**4. 窗体结构优化**
- 移除系统标题栏，自定义透明标题栏
- 精简窗口控制按钮（仅保留关闭）
- 标题栏可拖拽移动窗口
- 禁用窗口阴影，减少视觉干扰

**5. 粉色主题全面升级**
- 配色：温柔玫瑰粉 + 薰衣草紫 + 蜜桃橘
- 背景：5色梦幻渐变
- 卡片：白粉渐变 + 粉色阴影 + 悬停效果
- 滚动条/按钮/输入框：粉色渐变样式
- 主题名称改为"甜美粉"

**6. 励志句子扩充**
- 通用名言：50+ 条
- 女生风格名言（粉色主题专属）：55+ 条
- 支持 emoji 表情

### 🔧 功能优化

**7. 截止日期设置优化**
- 改进日期选择器交互
- 更便捷的日期设置方式

### 🐛 Bug 修复

- 修复透明模式下白边问题
- 修复主题切换时透明度残留
- 修复 Rust 端未使用变量警告
- 修复 base64 解码 deprecated 警告
